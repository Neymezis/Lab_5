# TaskFlow - Менеджер задач

TaskFlow - это кроссплатформенное приложение для управления задачами и проектами на русском языке. Приложение помогает пользователям организовывать свои задачи, устанавливать приоритеты и отслеживать прогресс выполнения.

# Sprint Review
Завершённые задачи:
• Issue #1: "Создать базовую структуру проекта .NET MAUI"
Создана структура проекта, настроены пакеты, проект компилируется для всех платформ.

• Issue #2: "Реализовать модель данных для задач"
Создан класс Task с валидацией и методами работы, написаны unit-тесты.

• Issue #3: "Разработать главный экран приложения"
Реализован MainPage.xaml с CollectionView, фильтрацией, сортировкой и навигацией.

Доставленные результаты:
• Рабочий каркас приложения .NET MAUI

• Модель данных для задач с тестами

• Функциональный главный экран с базовым UI

• Возможность отображения, фильтрации и сортировки задач

Незавершённые задачи:
• Issue #5: "Настроить локальную базу данных SQLite" (в процессе)
Частично реализован репозиторий, требуются доработки миграций и тестирование.

• Issue #8: "Реализовать экран деталей задачи" (не начата)
Задача переносится в следующий спринт.

# Sprint Retrospective
Что прошло хорошо:
• Эффективное планирование и соблюдение приоритетов

• Все high-priority задачи завершены вовремя

• Соблюдение критериев Definition of Done

• Хорошее качество кода и написание тестов

Что пошло не так: 
• Недооценка сложности работы с базой данных

• Перегрузка разработчиков из-за амбициозного плана

• Отсутствие резервного времени на непредвиденные сложности

• Позднее начало работы над сложными задачами

Что улучшить в следующем спринте:
• Более реалистичное планирование с учётом рисков

• Резервировать 20% времени на непредвиденные задачи

• Начинать сложные задачи в начале спринта

• Внедрить обязательный code review
